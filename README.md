# Federation Manager Log Export to Azure Sentinel Function

This is a source that can be provided straight to an azure function to be able to pull yan Identity Providers logs into your sentinel instance.

## How to use

Please see the tutorial found over at <https://tutorials.aaf.edu.au/exporting-identity-provider-logs/04-azure-function-implementation>

## How to build locally

At the moment there is no running locally, But all the release does currently is

* cd src
* npm install
* zips src folder
